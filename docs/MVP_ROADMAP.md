# MVP And Roadmap

## MVP Strategy

The MVP should prove that creators want to operate an AI character vlog channel repeatedly, not just generate one novelty video.

The first version should optimize for:

- Fast first character setup
- Fast first vlog episode
- Strong character continuity
- Export-ready short-form package
- Simple feedback loop for the next episode

## MVP Scope

### Must Have

- Character profile creation
- Persona JSON/version storage
- 10 vlog templates
- Script and scene generation
- Caption and hashtag generation
- Thumbnail concept generation
- Export package for vertical short-form video
- Manual publishing checklist
- Basic performance tracking

### Should Have

- Character visual reference upload
- Simple review/approval workflow
- AI disclosure injection
- Episode calendar
- Basic Fanvue content plan export

### Could Have

- YouTube upload integration
- TikTok upload integration
- Instagram Reels upload integration
- Fanvue post draft creation
- Fan memory CRM

### Not In MVP

- Full autonomous DM replies
- Full social network
- Native video editor with timeline-level control
- Marketplace for characters
- Multi-agent creator studio

## Suggested Build Phases

### Phase 0 - Validation

Duration: 1-2 weeks

- Create landing concept
- Interview 10-20 target creators
- Produce 3-5 sample AI character vlog series
- Validate whether users care about repeat publishing

### Phase 1 - Creator Workflow MVP

Duration: 4-6 weeks

- Character profile
- Vlog templates
- Episode generator
- Export package
- Manual upload workflow
- Basic dashboard

### Phase 2 - Publishing Integrations

Duration: 4-8 weeks

- YouTube Shorts integration first
- TikTok and Instagram Reels after app review readiness
- Publishing status tracking
- Platform-specific metadata rules

### Phase 3 - Fan Monetization

Duration: 4-8 weeks

- Fanvue integration
- Paid content planning
- PPV asset packaging
- Fan segments
- Human-reviewed DM drafts

### Phase 4 - Character Operating System

Duration: ongoing

- Fan memory
- Story arcs
- Character version performance analytics
- Multi-character management
- Agency workflows

## Key Risks

- AI video quality is not consistent enough for repeated content.
- Character face and voice drift breaks fan trust.
- Platforms penalize low-quality or spam-like AI uploads.
- AI disclosure is handled inconsistently.
- Users want one-off novelty outputs but do not become retained creators.
- Publishing API reviews slow down integration roadmap.

## Risk Mitigations

- Start with SFW short-form content.
- Keep human review in the publishing loop.
- Add disclosure by default.
- Start with export packages before full auto-publishing.
- Track repeat episode creation as the main activation signal.
- Store character versions and output audit logs from day one.
